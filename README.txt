PlaceFlow - Simple Project Guide

1) How to run this project

Step 1: Install Node.js on your computer.
Step 2: Open this project folder in terminal.
Step 3: Go to frontend folder:
cd frontend
Step 4: Install packages:
npm install
Step 5: Start project:
npm run dev
Step 6: Open browser:
http://localhost:5173


2) What this project does

PlaceFlow is a placement management system.
It helps college placement work in one place.
It has two roles:
- Admin (TPO)
- Student

Admin can manage companies and student progress.
Students can apply and track their status.


3) How to use

Admin side:
- Open Migration page
- Upload CSV or add Google Sheet link
- View dashboard charts and student data
- Add new company from Add Company page
- Update application status in workflow order

Student side:
- Open Student page
- View ongoing companies
- Click Apply if eligible
- Track status in status tracker


4) Notes

- This project runs in demo mode.
- No real login is required.
- Data is saved in browser local storage.
- You can later connect Firebase if needed.
